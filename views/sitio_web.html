Modulo - Sitio Web

Este módulo permite crear y gestionar una página web profesional sin necesidad de saber programar.

¿Cómo se usa?

  • Se accede desde el panel a Sitio Web.
  • Podemos usar el editor visual para arrastrar bloques, imágenes, textos, botones, etc.
  • Es posible enlazar el sitio con la tienda online y el blog.
  • Se pueden añadir formularios de contacto, chat en vivo y optimizar el SEO.
